Arrays

An array is a special variable, which can hold more than one value at a time.

If you have a list of items (a list of car names, for example), storing the cars in single variables could look like this:

            var car1 = "Saab";
            var car2 = "Volvo";
            var car3 = "BMW";

However, this is not efficient. If you want to store more cars, you have to create new variables for each car:

            var car4 = "Toyota";
            var car5 = "Honda";

To solve this problem, JavaScript provides arrays.

An array can hold many values under a single name, and you can access the values by referring to an index number.

Creating an Array

In JavaScript, arrays are written with square brackets [].

            var cars = ["Saab", "Volvo", "BMW"];

Accessing Array Elements

You access an array element by referring to the index number.

            var cars = ["Saab", "Volvo", "BMW"];
            var firstCar = cars[0]; // Saab

Modifying Array Elements

You can change an array element by referring to the index number.

            var cars = ["Saab", "Volvo", "BMW"];
            cars[0] = "Toyota"; // Now the first car is Toyota

Array Properties and Methods

Arrays come with a set of built-in properties and methods. For example, you can use the length property to get the number of elements in an array:

            var cars = ["Saab", "Volvo", "BMW"];
            var numberOfCars = cars.length; // 3

And you can use methods like push() to add new elements to the end of an array:

            var cars = ["Saab", "Volvo", "BMW"];
            cars.push("Toyota"); // Now cars is ["Saab", "Volvo", "BMW", "Toyota"]

Looping Through an Array

You can loop through the elements of an array using a for loop:

            var cars = ["Saab", "Volvo", "BMW"];
            for (var i = 0; i < cars.length; i++) {
                console.log(cars[i]);
            }

This will print each car name to the console.

Arrays are a powerful way to store and manage collections of data in JavaScript!

Here are some common array methods:

  • push() - Adds an element to the end of the array
  • pop() - Removes the last element from the array
  • shift() - Removes the first element from the array
  • unshift() - Adds an element to the beginning of the array
