Methods and Loops in Programming

Methods (or functions) are blocks of code that perform a specific task. They help in organizing code, making it reusable, and improving readability. Loops are used to execute a block of code multiple times based on a condition.

Example of a Method:

function addNumbers(a, b) {
    return a + b;
}

Example of a Loop:

for (let i = 0; i < 5; i++) {
    console.log(i);
}
